# express-post-json-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does the `express.json()` middleware do and when would you need it?
  The express.json() middleware in Express.js is used to parse incoming request bodies in JSON format. It is part of the body-parser module that helps parse various types of request bodies.
